Propel

API Ref Notebook Github

Use it in Node:

npm install propel
import { $ } from "propel";

Use it in a browser:

<script src="http://propelml.org/propel-2.99.1.js"></script>

Propel aims to provide a GPU-backed numpy-like infrastructure for scientific computing in JavaScript. JavaScript is a fast dynamic (optionally typed) language which, we think, could be an ideal workflow for statisticians of all flavors.

Propel runs run both in the browser and natively from Node. In both environment Propel is able to use GPU hardware for computations. In the browser it utilizies WebGL through DeepLearn.JS and on Node it uses TensorFlow's C API.

Propel embraces imperative style ML. To achieve this, Propel provides a general purpose "gradient function" (implemented with backprop) on which all sorts of variants of stochastic gradient decent are built.
