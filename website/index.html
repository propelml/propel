Propel

Mathematics for JavaScript

API Ref Notebook

In 2018, JavaScript is a first class language. It runs everywhere, it outperforms other dynamic languages, and has a beautiful modern type system.

JavaScript deserves to have a first class mathematics framework. This project intends to deliver that, with a special emphasis on machine learning.

Use it in Node:

npm install propel import { $ } from "propel";

Use it in a browser:

        <script src="propel-1.js"></script>

Propel provides a familiar API to those used to doing scientific programming in Python. However Propel is carefully crafted to the JavaScript environment.

Propel runs run both in the browser and natively from Node. In both environment Propel is able to use GPU hardware for computations. In the browser it utilizies WebGL and on Node it uses TensorFlow's C API.

Propel embraces imperative style ML. To achieve this, Propel provides a general purpose "gradient function" (implemented with backprop) on which all sorts of variants of stochastic gradient decent are built.
